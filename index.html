What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the page structure as a tree where each node is an object representing part of the document (like elements or text).

How do you select an element from the DOM?

I can select elements using methods like `document.getElementsById`, `document.querySelector`, or `document.getElementByClassName`.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allow you to handle events at a higher level (like a parent element), improving effeciency, especially, when dynamically adding elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate attributes using methods like`setAttribute`, and styles using `element.style` to directly change CSS properties.
